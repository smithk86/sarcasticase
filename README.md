### Usage

#### install
pip install sarcasticase

#### command line
```sh
$ sarcasticase i know you are but what am i
> i KnOw YoU aRe BuT wHaT aM i
```

#### import function
```sh
from sarcasticase import sarcasticase
sarcastic_text = sarcasticase("i know you are but what am i")
```
